# dplooter


```
usage: dplooter.py [-h] [-t TARGET] [-u USER] [-p PASSWORD] [-d] [-v] [-e EXTENSION]

Fuzz Distribution Point packages via HTTP(s)

options:
  -h, --help            show this help message and exit
  -t TARGET, --target TARGET
                        Target DP IP or hostname.
  -u USER, --user USER  Site Server Admin Username.
  -p PASSWORD, --password PASSWORD
                        Site Server Admin Password or Hash (LMHASH:NTHASH
  -d, --download        Download files found from fuzzing
  -v, --verbose         Enable verbose output
  -e EXTENSION, --extension EXTENSION
                        filter for file extension. Comma separated list ex: 'ps1,exe,dll'

```
